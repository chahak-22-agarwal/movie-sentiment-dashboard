# Movie Sentiment Analysis Dashboard

A Machine Learning-based web application that predicts whether a movie review is positive or negative using Natural Language Processing (NLP) and Scikit-learn.

## Features
- Real-time sentiment prediction
- Interactive dashboard interface
- Machine Learning model integration
- TF-IDF vectorization
- Flask backend support
- Trained on IMDB movie review dataset

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS
- Machine Learning
- NLP

## How It Works
1. User enters a movie review.
2. The review is processed using TF-IDF vectorization.
3. The trained ML model predicts sentiment.
4. Result is displayed as Positive or Negative.
